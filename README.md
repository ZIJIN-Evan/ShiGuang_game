## 游戏特点

画风唯美，大面积色块来凸显主角渺小，人物闯关收集图腾。

## 核心玩法

通过闯关搜集不同图腾的图鉴，通过不同的组合来获得不同的能力组合，以此来更方便的通过下一关。
